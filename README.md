# news_site_parsing_uni — парсер новостей

Этот проект для университета выполняет парсинг новостной страницы, сохраненной из веб-архива:  
https://web.archive.org/web/20230903112115/https://iz.ru/news  

Программа извлекает заголовки новостей и ссылки, а затем группирует их по тематическим разделам  
(Общество, Мир, Здоровье, Спорт и др.).

# Зависимости
Перед запуском установите необходимые библиотеки: 
  - pip install -r requirements.txt

Файл requirements.txt содержит:
  - beautifulsoup4

# Как запустить
1. Скачайте страницу из веб-архива и сохраните ее как **iz_news.html** в корень проекта.
2. Запустите основной скрипт:
   python parse_news.py

После запуска программа выведет новости в формате словаря (JSON), сгруппированные по разделам.
